## Developer Setup
### `node v8.5.0`
### `npm v5.4.0`
### Recommended to install Node Version Manager https://github.com/creationix/nvm

## Available Commands

### `yarn i` or `npm i`
Installs dependencies

### `yarn run start` or `npm run start`
Run the app on localhost

### `rm -rf lib && rm -rf dist && rm -rf node_modules`
Clean (as needed)
